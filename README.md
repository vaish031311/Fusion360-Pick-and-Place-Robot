# 3-DOF Robotic Pick and Place Arm
**Tool:** Autodesk Fusion 360 | **Domain:** Robotics & Mechanical Design

---

## Overview
Designed and simulated a 3-DOF robotic pick and place arm in Autodesk Fusion 360. The project covers complete parametric part modelling, mechanical assembly with revolute joint relationships, and kinematic motion simulation of a full pick and place cycle.

---

## Specifications

| Parameter | Details |
|---|---|
| Degrees of Freedom | 3-DOF |
| Joint 1 — Base | Revolute — Z axis (sweep left/right) |
| Joint 2 — Elbow | Revolute — Y axis (bend up/down) |
| Joint 3 — Gripper | Rigid joint |
| Gripper Type | Claw style |
| Software | Autodesk Fusion 360 |
| Units | Millimetres |
